## 🛠️ 구축된 시스템

### 1. 데이터 검증 모듈 (`01_validate_data.py`)
**기능**:
- 데이터 로드 및 스키마 검증
- 결측치 자동 감지
- 이벤트 전적 자동 필터링
- 멤버 이름 자동 추출 (오타 방지)
- 기본 통계 산출

**출력**:
- `validation_result.json` - 검증 결과
- `validated_data.csv` - 검증된 데이터
- `member_names.json` - 멤버 이름 목록

### 2. 팀 분석 모듈 (`02_analyze_team.py`)
**기능**:
- 전체 통계 (승률, 경기 수)
- 월별/분기별 추이 분석
- 종족별 성과 분석
- 티어별 분포 분석
- 경기 유형별 비교 (스폰 vs 대회)
- 맵별 승률 (상위 20개)
- 종족 상성 매트릭스 (9x9)

**출력**:
- `team_analysis.json` - 팀 분석 결과

### 3. 멤버별 분석 모듈 (`03_analyze_members.py`)
**기능** (15명 전원):
- 개인 전체 통계
- 종족별/티어별 세부 전적
- 경기 유형별 성과
- 월별 추이 분석
- 종족 상성 분석
- 맵별 승률 (상위 10개)
- 강점/약점 자동 파악
- 성장 곡선 (전반기 vs 후반기)

**출력**:
- `member_analysis.json` - 멤버별 분석 결과

### 4. 차트 생성 모듈 (`04_generate_charts.py`)
**기능**:
- 기존 보고서 스타일 계승 (어두운 배경)
- 7종 차트 자동 생성
- 한글 폰트 지원
- 고해상도 PNG 출력 (150 DPI)

**출력 차트**:
1. 멤버별 전적 비교 (막대 + 승률 라인)
2. 월별 전적 추이
3. 종족별 성과 비교
4. 종족 상성 히트맵
5. 티어별 분포
6. 경기 유형별 비교
7. TOP 성과자 (100경기 이상 기준)

### 5. 원클릭 실행 스크립트
**파일**:
- `run_report.py` - Python 마스터 스크립트
- `run_report.bat` - Windows 배치 파일
- `README.md` - 사용 설명서

**실행 시간**: **약 5~6초** (전체 파이프라인)

- 모든 분석과 차트에 일관되게 적용

### 5. 기존 스타일 계승

#### 방법 2: Python 자동 생성 (추가 개발 필요)

- 템플릿 기반 자동 조립
- 소요 시간: 추가 2~3시간 개발

#### 방법 3: Markdown → PDF
- Markdown 보고서 작성
- `weasyprint` 또는 `pandoc`로 PDF 변환
- 소요 시간: 추가 1~2시간 개발
